## PyQt GUI 요약

### 실행 명령

```bash
pip install -r requirements-dev.txt
python -m src.boundary.screen.app
# 또는
python -m src.boundary.screen
```

### 화면 구성

| 요소 | 설명 |
|------|------|
| 4×4 `QSpinBox` | 0~16 (0 = 빈 칸, `·` 표시) |
| **Validate** | `InputValidator.validate_grid()` 호출 |
| **Clear** | 전체 칸 0으로 초기화 |
| **Result** | `[INVALID_SIZE] Grid must be 4x4.` 또는 형상 통과 안내 |

### ECB 준수

- GUI는 **Boundary** (`src/boundary/screen/`)에만 위치
- `InputValidator` 호출; Control/Entity에 PyQt import 없음
- `GRID_SIZE` 등 상수는 `src/boundary/constants.py` SSOT 사용

### GUI와 GREEN 범위 매핑

| 입력 | GUI 동작 |
|------|----------|
| `grid=None` | GUI 해당 없음 (항상 4×4 구조) |
| `grid=[]` | GUI 해당 없음 (pytest/REPL로 확인) |
| 4×4 격자 | Validate → 형상 통과 안내 (`NotImplementedError` catch) |
